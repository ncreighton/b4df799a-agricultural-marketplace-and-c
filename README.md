# Agricultural Marketplace and Commodity Trading Discord Bot

A Discord bot for farming cooperatives, agricultural commodity traders, and rural market groups. Posts daily commodity spot price updates for grains, livestock, and produce from USDA and CME Group feeds, provides weather impact alerts for growing regions, facilitates buy and sell offer posting between members, tracks price trends with weekly chart summaries, and sends planting season reminders and harvest deadline notifications tailored to each member's crop profile.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
